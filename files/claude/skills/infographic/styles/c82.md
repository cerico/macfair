# Style: c82

The default aesthetic. Draws from Nicholas Rougeux's work: classical typography, structured grids, muted naturalist palettes. Think well-designed book page, vintage scientific illustration, or museum exhibit panel.

## Typography
- **Headings:** Georgia or Garamond — serif, classical weight
- **Body:** Helvetica, "Helvetica Neue", sans-serif — clean, readable
- **Stats:** Large serif numbers (Georgia, 48-60px) — typographic emphasis, not colored boxes
- **Labels/meta:** Small caps feel, letter-spacing: 2px, uppercase, 11-12px

## Color Palette

Primary palette — muted, naturalist:
- Canvas: #F5F0E8 (warm cream)
- Text primary: #2C2C2C (near-black, warm)
- Text secondary: #6B5E50 (warm gray-brown)
- Accent 1: #2D4A3E (deep forest green)
- Accent 2: #7A3B2E (burgundy/rust)
- Accent 3: #1E3A5F (deep navy)
- Accent 4: #8B7A2E (aged gold/ochre)
- Rule lines: #C4B8A8 (warm light gray)
- Highlight bg: #EDE7DA (lighter cream)

## Layout
- 800x1100px canvas
- 60px margins on all sides
- Content width: 680px
- Vertical rhythm:
  - Section heading to content: 24px
  - Content to stats: 28px
  - Stats to divider: 28px
  - Divider to next section heading: 36px
  - Text-only section to divider: 32px
- Classical proportions — header takes ~15% of height

## Visual Elements
- **Border frame:** 1px rule line (#C4B8A8) inset 24px from edges
- **Section dividers:** Thin horizontal rules (0.5px) with optional center ornament (—✦—)
- **No colored accent bars** — use typography and whitespace instead
- **Stats:** Large serif number above small-caps label, separated by thin rule
- **Header:** Title centered, subtitle below in italic serif
- **Footer/takeaway:** Set apart with double rule lines above, italic text
- **Formula/code block:** `#EDE7DA` background rect spanning content width (680px), 16px padding inside. Georgia italic, 13px, secondary text color (#6B5E50). Use `<tspan>` with `dy="22"` between lines.
- **Flow diagram:** Items evenly distributed across content width using `text-anchor="middle"`. Arrow characters (`→`) between items in muted color (#C4B8A8, 16px). Sublabels directly below in smaller muted text (11px, #6B5E50), same x-anchor.

## Notes
- **No shadows, no gradients, no rounded corners** — clean, flat, classical
- Section heading color: rotate through accent colors (green, burgundy, navy, gold)
- Swatch visibility: when a color swatch is similar to the canvas, add `stroke="#C4B8A8" stroke-width="0.5"`
